Traceable Reference Blocks

Fact Cards You Can Trust

Ever shared a fact online and wondered "where did this come from?" TREF solves that. It's like a trading card for facts - with proof built in.

Make a Fact Card How It Works

What's a TREF Block?

1

Your Fact

Write down something you know - a quote, a statistic, a recipe, anything. This is what you want to share.

2

Your Source

Add where you learned it. A website, a book, a video. This helps others trust your fact.

3

A Fingerprint

TREF creates a unique code from your fact. If anyone changes it, the code won't match anymore. Proof it's real!

4

Share Anywhere

Drag it, copy it, send it. The sources come along for the ride. No more "where did you get that?"
