What is TREF?

Content-Addressed

Every block has a unique ID derived from its content using SHA-256. Change the content, change the ID. Verify integrity instantly.

Self-Contained

A .tref file contains everything: content, metadata, license, references, and lineage. No external dependencies.

Traceable Lineage

Derive new blocks from existing ones. The parent chain is preserved, so you always know where knowledge came from.

License Preserved

Attribution travels with the content. CC-BY-4.0 by default. Original creators always get credit.

Quick Start

Create a block

Write your knowledge and publish it as a TREF block.

tref publish "The speed of light is 299,792,458 m/s"

Share it

The block gets a unique ID. Share the .tref file or embed it in your page.

sha256:7a4f2c... â†’ 7a4f2c...tref

Verify & Derive

Anyone can verify the block's integrity. Derive new blocks while preserving lineage.

tref validate block.tref
tref derive sha256:7a4f2c... "Updated with source"
