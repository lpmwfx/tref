Traceable Reference Format

Knowledge blocks you can verify, share, and trust. Content-addressed, self-contained, and always traceable to source.

Build a Block Developer Guide

What is TREF?

1

Content-Addressed

Every block has a unique ID derived from its content using SHA-256. Change the content, change the ID. Verify integrity instantly.

2

Self-Contained

A .tref file contains everything: content, metadata, license, references, and lineage. No external dependencies.

3

Traceable Lineage

Derive new blocks from existing ones. The parent chain is preserved, so you always know where knowledge came from.

4

License Preserved

Attribution travels with the content. CC-BY-4.0 by default. Original creators always get credit.
